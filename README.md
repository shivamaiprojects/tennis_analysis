# 🎾 Tennis Analysis (Computer Vision Project)

This project uses computer vision techniques (YOLO + OpenCV) to analyze tennis matches.

## 🚀 Features (Planned)

- Player detection
- Tennis ball tracking
- Court line detection
- Player movement tracking
- Speed and shot analysis

## 🛠️ Tech Stack

- Python
- OpenCV
- Ultralytics YOLO
- PyTorch
- NumPy
